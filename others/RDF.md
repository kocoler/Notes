RDF
===

资源描述框架(Resource Description Framework，简称RDF)

RDF由RDF Data Model、RDF Schema和RDF Syntax三个部分组成。

RDF Data Model: 资源(主) 属性(谓) 值(宾)
RDF Schema: 定义描述资源
RDF Syntax: XML
