MAC
===

#### **应用文件位置**：

  - **Configuration**

    ```
    ~/Library/Preferences/
    ```

  - **Caches**

    ```
    ~/Library/Caches/
    ```

  - **Plugins**

    ```
    ~/Library/Application Support/
    ```

  - **Logs**

    ```
    ~/Library/Logs/
    ```
